This document records errata and corrections suggested by the community in the specifications published by the W3C CSV on the Web Working Group on the 17 December 2015.

Introduction

These corrections have no normative status as they have not been incorporated in the specification through the Recommendation Track or Working Group Note process. If you find an issue in one of the documents listed below that you believe may be an error and wish to have it recorded here, please raise an issue on the issue list of the group‘s GitHub repository using the label Errata. (If possible, you should also use the available labels to point to the document(s) where you found a problem.) The issue may be discussed by the community; if the discussion gets to an stable state, and the error is accepted, the issue should be recorded on this page under the relevant section for the respective document(s), giving a short summary of the nature of the error and the proposed solution. This can be done by anyone involved, issuing a pull request on GitHub.

If you have problems following this process, but you want nevertheless report an error, you can also contact the staff contact of the Working Group, Ivan Herman.

The list of open error issues can also be consulted on the issues’s list. These issues should remain open until the changes are incorporated into newer editions of the documents, at which time they are closed.
